# TehGM's WE Wallpapers

Note: these wallpapers were fully generated by Claude Fable 5 and Claude Opus 4.8, as an experiment and for fun. The idea initially was merely to have a generated black hole wallpaper (to use with Wallpaper Engine). However I ended up liking the result so much so decided to share it, especially since it can be statically hosted. Later I figured I could get Claude to build several more wallpapers under the single project.

I didn't touch any code in the project. Doing such cool looking things is out of my competences. I am a back-end dev, dammit!

Live at https://wallpapers.tehgm.net

---

A collection of statically hosted animated wallpapers, built for
[Wallpaper Engine](https://www.wallpaperengine.io/) but equally usable as a
browser background or screensaver.

## Wallpapers

| | Path | Status | Live |
|---|---|---|---|
| [Black Hole](blackhole/) | `/blackhole/` | Released | https://wallpapers.tehgm.net/blackhole/ |
| [Neonscape](neonscape/) | `/neonscape/` | Released | https://wallpapers.tehgm.net/neonscape/ |

Released wallpapers keep their saved links and scenes working across updates.

## Usage

Point Wallpaper Engine at the hosted URL, or open the `index.html` of any
wallpaper directly in a browser. Settings are exposed in an in-page panel and
persist via the URL.
